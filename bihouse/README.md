# BIHOUSE - Frontend Website

Website modern untuk coffee shop "BIHOUSE" yang dibangun menggunakan React, Vite, Tailwind CSS, dan Framer Motion.

## Fitur Utama

- **Desain Modern & Premium**: Menggunakan palet warna "Warm Cream" dan "Deep Espresso".
- **Dark Mode**: Dukungan tema gelap yang ramah di mata.
- **Animasi Interaktif**: Menggunakan Framer Motion untuk transisi yang halus pada menu dan hero section.
- **Responsif**: Tampilan yang optimal di desktop maupun mobile.
- **Support Bahasa**: Konten tersedia dalam Bahasa Indonesia.

## Teknologi yang Digunakan

- **[React](https://react.dev/)**: Library UI untuk membangun antarmuka pengguna.
- **[Vite](https://vitejs.dev/)**: Build tool yang cepat untuk pengembangan frontend modern.
- **[Tailwind CSS](https://tailwindcss.com/)**: Framework CSS utility-first untuk styling yang cepat.
- **[Framer Motion](https://www.framer.com/motion/)**: Library animasi untuk React.
- **[Lucide React](https://lucide.dev/)**: Koleksi icon yang indah dan konsisten.

## Cara Menjalankan Project (Cloning)

Ikuti langkah-langkah berikut untuk menjalankan project ini di komputer Anda:

1.  **Clone Repository**
    ```bash
    git clone https://github.com/username/bihouse.git
    cd bihouse
    ```

2.  **Install Dependencies**
    Pastikan Anda sudah menginstall [Node.js](https://nodejs.org/). Kemudian jalankan perintah:
    ```bash
    npm install
    ```

3.  **Jalankan Development Server**
    Untuk memulai server lokal:
    ```bash
    npm run dev
    ```
    Buka browser dan akses `http://localhost:5173`.

4.  **Build untuk Production**
    Jika ingin membuat versi aplikasi yang siap deploy:
    ```bash
    npm run build
    ```
    Hasil build akan berada di folder `dist`.

## Struktur Project

- `src/components`: Komponen UI seperti Navbar, Hero, Menu, Footer.
- `src/App.tsx`: Komponen utama aplikasi.
- `public`: Aset statis (gambar menu, icon, dll).
